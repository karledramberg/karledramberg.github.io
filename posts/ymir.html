Ymir

Ymir is an ongoing project of mine to learn procedural generation techniques. For now, Ymir isn't finished. It serves it's purpose of providing a platform for experimentation, but there are no plans for it to become a full-blown application.

So far I've implemented Perlin and Open Simplex Noise and have used Voronoi Polygons for continent simulation.

Ymir also can interpret the generated world in different ways for different types of maps. For example elevation can be shown with color or topography.

Examples of maps using Perlin noise


Ymir is currently written in Go and previously in Java. I switched languages both to learn Go and move away from OOP.

Ymir currently lives as a web app in the browser.
You can find the source on GitHub.
