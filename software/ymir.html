Ymir Games
Journal About

Ymir is an ongoing project of mine to learn procedural generation techniques through map generation. For now, Ymir isn't finished and probably never will be. It serves it's purpose of providing a platform for experimentation, but there are no plans for it to become a full-blown application.

So far I've implemented Perlin and Open Simplex noise and have used Voronoi polygons to simulate continents.

Ymir can also interpret the generated world in different ways for different types of maps. For example elevation can be shown with topography or color.

Examples of maps using Perlin noise


Ymir is current written in Go and previously in Java. I switched languages both to learn Go and move away from OOP - the only programming paradigm I had used so far.

Ymir current lives as a web app in the browser.

You can find the source on my GitHub page.
